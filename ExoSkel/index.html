About ExoSkel

ExoSkel is an enhancement to Skeleton CSS Boilerplate which is a small collection of well-organized CSS files that can help you rapidly develop sites that look beautiful at any size, be it a 17" laptop screen or an iPhone. It's based on a responsive grid, but also provides very basic CSS for typography, buttons, forms and media queries. Go ahead, resize this super basic page to see the grid in action.

ExoSkel adds a 1280px layout for going beyond the 960px grid supported by skeleton. ExoSkel also builds in sticky footers (example provided on this page) and a "fullWidth" class that can be used quickly to use 100% of the browsers width (see the HR above?).

Three Core Principles

Skeleton (and therefore ExoSkel) is built on three core principles:

  • A Responsive Grid Down To Mobile: Elegant scaling from a browser to tablets to mobile.
  • Fast to Start: It's a tool for rapid development with best practices
  • Style Agnostic: It provides the most basic, beautiful styles, but is meant to be overwritten.
